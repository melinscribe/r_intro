Variance

Definition:
Variance measures how far each value in a dataset is from the mean.
Itâ€™s calculated as:

\(\text{Variance} = \frac{\\sum (x_i - \bar{x})^2}{n - 1}\)

Where:

  • \(x_i\) = each observation
  • \(\bar{x}\) = mean
  • \(n\) = number of observations

Key points:

  • Larger variance â†’ values are more spread out.
  • Small variance â†’ values cluster around the mean.
  • Units: squared (e.g., USDÂ²), which is why we often use the standard deviation (square root of variance) for interpretation.

Interquartile Range (IQR)

Definition:
The IQR measures the range covered by the middle 50% of the data.
Itâ€™s the difference between:

  • Q3 (75th percentile) = value below which 75% of data fall
  • Q1 (25th percentile) = value below which 25% of data fall

\(\text{IQR} = Q3 - Q1\)

Key points:

  • Robust to outliers â€” ignores the extreme top and bottom 25%.
  • Great for skewed data (like fatalities in conflict).

Skewness

Definition:
Skewness measures the asymmetry of a distribution.

  • Skewness = 0 â†’ perfectly symmetric (bell curve)
  • Skewness > 0 â†’ right-skewed: long tail on the right (common in GDP and fatalities)
  • Skewness < 0 â†’ left-skewed: long tail on the left

Why it matters in conflict research:

  • Fatalities: Many observations are 0 or low, but a few wars cause extremely high counts â†’ right-skew.
  • GDP: Most countries are in the lower range, but a few are very rich â†’ right-skew.

Interpretation tip:

  • Right-skew (skewness > 0) is common in GDP/casualty data.
  • IQR is robust to outliers.

How to Read a Boxplot

A boxplot is a compact way to show: - Center (median) - Spread (IQR) - Tails (minimum & maximum within 1.5Ã—IQR) - Outliers (points beyond the whiskers)

Anatomy of a Boxplot

(Image Source Daniel Dauber)

  1. Box â†’ spans from Q1 (25th percentile) to Q3 (75th percentile)
    • IQR = Q3 âˆ’ Q1
  2. Horizontal line in box â†’ median (50th percentile)
  3. Whiskers â†’ extend to smallest/largest value within 1.5 Ã— IQR from the box
  4. Dots outside whiskers â†’ outliers (unusually high/low values)

Why Boxplots are Useful in Conflict Data

  • Show skewness visually:
    • If the median is closer to the bottom of the box â†’ right-skewed
    • If whiskers are uneven â†’ asymmetry
  • Highlight outliers:
    • Big wars or massacres will appear as dots far above the whisker
  • Compare groups side-by-side:
    • Conflict vs no-conflict
    • Different income groups

ğŸ”— 4) Relationships Between Variables

4.1 Correlation (Pearson/Spearman)

# Note: best is count; Spearman (rank) is robust
cor(data$gdp_pc, data$best,
    use = "complete.obs", method = "spearman")
[1] -0.2612842
  • method = "spearman" â†’ Use Spearmanâ€™s rank correlation, which:
    • Does not assume the variables are normally distributed
    • Measures how well the relationship between two variables can be described by a monotonic function (increasing or decreasing)
    • Uses ranks instead of actual values, making it robust to outliers and skewness.

How Spearmanâ€™s correlation works

  1. Rank all values of GDP per capita from lowest to highest.
  2. Rank all values of fatalities from lowest to highest.
  3. Compare the similarity of these rankings:
    • If rich countries tend to have more fatalities â†’ positive correlation
    • If rich countries tend to have fewer fatalities â†’ negative correlation
  4. Produce a correlation coefficient Ï between âˆ’1 and +1.
  • Value = âˆ’0.26 â†’ weak to moderate negative correlation
    â†’ As GDP per capita increases, fatalities tend to decrease (but not perfectly).
  • Magnitude:
    • 0.0 to Â±0.3 â†’ weak
    • Â±0.3 to Â±0.6 â†’ moderate
    • Â±0.6 to Â±1.0 â†’ strong
  • Sign:
    • Positive â†’ variables tend to move in the same direction
    • Negative â†’ variables tend to move in opposite directions

Always check a scatterplot to visually confirm:
